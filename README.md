# gitlab-webhooks.cr

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     gitlab-webhooks.cr:
       github: chussenot/gitlab-webhooks.cr
   ```

2. Run `shards install`
